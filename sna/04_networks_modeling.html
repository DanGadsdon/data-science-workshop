ergm package works with object in network format (not igraph), so we need to convert first between packages using the intergraph package.

Edges = number of edges in the network. Similar to a constant. exp(edges) close to observed density. Mutual = number of mutual edges in the network. If positive, it means reciprocity is more common than expected. If you see an edge from i to j; then you are exp(mutual) times more likely to see an edge from j to i as well. nodematch = probability that an edge between nodes with the same value on this variable exists. exp(nodematch) times more likely for an edge to exist when values match than otherwise.
